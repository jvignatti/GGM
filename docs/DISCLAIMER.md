# Disclaimer
## Legal and Organizational Notice

---

## Data Ownership

All data processed, displayed, and generated by this system belongs to the **State of Vermont**. This includes but is not limited to:

- Grant records, award amounts, and expenditure data
- Invoice and quarterly report records
- Supervisor and agency assignments
- Any derivative calculations, scores, or summaries generated from the above

The dashboard and pipeline tools are operational utilities built to support internal program management. They do not create, modify, or have any legal standing over the underlying data.

---

## Intended Use

This dashboard and its associated pipeline are intended solely for **internal use by authorized staff** of the Highway Safety Office. Use of this tool does not constitute an official financial report, audit document, or legally binding record.

All figures displayed in the dashboard are derived from source data exported from the grants management portal. In case of discrepancy, the grants portal is the authoritative source of record.

---

## Data File Distribution

> **The `grants_data.json` file must not be shared outside the organization.**

Users who access or receive the `grants_data.json` file are responsible for:

- Keeping the file on organization-managed devices and systems
- Not distributing the file to external parties, vendors, or contractors without explicit authorization
- Not uploading the file to public repositories, cloud storage accessible to the public, or external communication platforms
- Reporting any accidental disclosure to their supervisor and IT immediately

The HTML dashboard file (without the JSON) may be shared publicly and contains no sensitive information.

---

## Accuracy and Limitations

This tool is provided as an operational aid. The following limitations apply:

- **The dashboard reflects a snapshot in time.** Data is only as current as the last pipeline run.
- **Scoring and signal logic are operational heuristics,** not official compliance determinations.
- **Portal timeouts and scraping errors** may result in incomplete data for specific grants. The pipeline logs these cases.
- **Manual submissions outside the portal** (e.g., email-submitted invoices) are not captured by the scraper and will not appear in the dashboard.
- **This tool does not replace professional judgment.** Staff should verify critical decisions against the authoritative portal.
